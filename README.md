# SaleMove Console App

Welcome!

This app allows creating Engagements using just a terminal.

![Sample session](assets/example.png?raw=true "Sample session")

## Usage

### Using ruby

```ruby
bundle
bundle exec ./start
```

### Using docker

```sh
docker build -t console-app .
docker run -ti console-app
./start
```


### Keybindings

Use `CTRL+C` to exit.
